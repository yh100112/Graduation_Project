# Project Title

Calorie Guide

## Motivation

코로나 펜데믹 현상으로 인해 체중이 증가한 사람이 증가함으로 인해 현대인의 식단관리를
스스로 수월하게 관리할 수 있는 어플을 만들면 어떨까 하는 생각으로 주제 선정
